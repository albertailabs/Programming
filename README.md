# 프로그래밍을 위한 컴퓨터언어 저작도구

알버트 코딩 SW를 설치하면 스크래치와 엔트리, 플레이봇, 자바스크립트를 사용할 수 있습니다.


시작하기 전
--
- USB 동글의 디바이스 드라이버는 설치 파일에 포함되어 있으며, 설치 과정에서 디바이스 드라이버도 같이 설치됩니다.
- USB 동글을 PC에 연결하기 전에 알버트 코딩 소프트웨어를 먼저 설치해야 합니다.
